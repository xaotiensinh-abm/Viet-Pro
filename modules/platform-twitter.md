# Module H5: Platform — Twitter/X Thread

> **Load**: Khi target platform là Twitter/X (tweet đơn hoặc thread)
> **Knowledge**: `knowledge/social-media-vn-2025.md`

---

## Nhiệm vụ

Viết tweet/thread gọn gàng, ý tưởng sắc bén, tối đa engagement trong giới hạn ký tự.

## Twitter/X Content Rules

### Format
```
- Tweet đơn: ≤280 ký tự (tiếng Việt ~100-140 từ)
- Thread: 5-15 tweets, mỗi tweet = 1 ý hoàn chỉnh
- Media: Ảnh/video tăng engagement 2-3x
- Hashtag: 1-3 maximum, tích hợp tự nhiên
- Mentions: Tag relevant khi cần
```

### Tweet Đơn — Hook Patterns

```
Pattern 1: Hot take
  → "Unpopular opinion: CV 2 trang tốt hơn 1 trang. Fight me."

Pattern 2: Thread tease
  → "Tôi mất 3 năm học điều này. Bạn mất 1 phút đọc. 🧵"

Pattern 3: Observation
  → "Sáng nào cũng thấy LinkedIn đầy 'chúc ngày mới tốt lành'. Ai đọc?"

Pattern 4: Listicle micro
  → "3 thứ tôi ước ai nói sớm:
     1. ...
     2. ...
     3. ..."

Pattern 5: Question provocative
  → "Bạn có work-life balance không? Hay đang tự lừa mình?"
```

### Thread Architecture

```
Tweet 1: HOOK + "🧵" signal
  → Compelling opening, promise value, indicate thread
  → Ví dụ: "Tôi từ 0 đến 10k followers trong 90 ngày. Đây là playbook. 🧵"

Tweet 2-3: CONTEXT / SETUP
  → Background, why this matters, credibility
  → Self-contained — ai đọc random tweet cũng hiểu

Tweet 4-N: BODY — Core content
  → Mỗi tweet = 1 insight/step/lesson
  → Numbered: "3/" hoặc "Step 3:"
  → Cliff giữa tweets: "Nhưng đây chưa phải phần hay nhất."

Tweet N-1: SUMMARY
  → Recap key points ngắn gọn
  → "TL;DR:" cho người scroll nhanh

Tweet N: CTA
  → "Retweet tweet đầu nếu hữu ích 🔄"
  → "Follow @handle cho thêm content kiểu này"
  → "Reply: Bạn sẽ thử cái nào đầu tiên?"
```

### Thread Template

```markdown
## Thread: [Chủ đề]

**1/** [HOOK — compelling, promise value] 🧵

**2/** [CONTEXT — tại sao topic này quan trọng]

**3/** [POINT 1 — insight + mini evidence]

**4/** [POINT 2 — insight + example]

**5/** [POINT 3 — insight + actionable tip]

**6/** [POINT 4 — contrarian hoặc surprise twist]

**7/** [POINT 5 — most important point, save for last]

**8/** TL;DR:
→ Point 1
→ Point 2
→ Point 3
→ Point 4
→ Point 5

**9/** [CTA]
Nếu thread này hữu ích:
- Retweet tweet 1 🔄
- Follow @handle cho insights hàng tuần
- Reply: bạn đang struggle ở bước nào?
```

### Engagement Techniques

| Kỹ thuật | Mô tả |
|----------|-------|
| Quote tweet bait | Viết opinion mạnh → người ta quote phản biện |
| Thread within thread | Link tới thread cũ liên quan |
| Poll companion | Đăng poll kèm thread để tăng interaction |
| Bookmark reminder | "Bookmark thread này — bạn sẽ cần sau" |
| Visual support | Tweet key kèm screenshot/infographic |

### Writing Rules cho Twitter

```
1. MỖI TWEET tự đứng — đọc riêng vẫn có giá trị
2. Câu NGẮN. Ý SẮC. Không rào đón.
3. Mỗi tweet chỉ 1 ý — không nhồi
4. Numbered threads dễ follow hơn
5. Line breaks tạo breathing room
6. Tránh: wall of text trong 1 tweet
```

## Anti-AI Rules cho Twitter

```
1. KHÔNG: "Let me share..." / "Cho phép tôi chia sẻ..."
2. PHẢI: Personality rõ — opinion, wit, edge
3. PHẢI: Mỗi tweet có punch — không filler
4. Tránh: Perfect parallel structure liên tục (quá đều = AI)
5. OK: Không hoàn hảo, viết tắt, informal
```
