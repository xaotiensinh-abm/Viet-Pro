# Module H3: Platform — TikTok / Reels

> **Load**: Khi target platform là TikTok, YouTube Shorts, Instagram Reels

---

## Nhiệm vụ

Viết caption/script ngắn, punchy, scroll-stopping cho short-form video platforms.

## TikTok Caption Rules 2025

### Format
```
- Độ dài: 50-150 từ (caption) / 30-60 giây (script)
- Tone: Conversational, Gen Z friendly, NHANH
- Emoji: OK nhưng không spam (3-7/bài)
- Hashtag: 5-10, mix trending + niche + branded
- Line breaks: Mỗi ý 1 dòng, scan siêu nhanh
```

### Scroll-Stopper Hooks (Câu đầu = 1 GIÂY)

```
Pattern 1: Challenge assumption
  → "Bạn skincare sai hết rồi."

Pattern 2: POV setup
  → "POV: Sếp hỏi 'có gì update không' mà bạn chưa làm gì 😭"

Pattern 3: Curiosity gap
  → "Mẹo này tiết kiệm 2 triệu/tháng mà ít ai biết."

Pattern 4: Relatable pain
  → "Khi bạn soạn presentation lúc 2h sáng..."

Pattern 5: Before/After tease
  → "Trước: 0 đơn. Sau 1 tuần áp dụng cái này: 47 đơn."

Pattern 6: "Đừng..." reverse psychology
  → "Đừng order Shopee nữa nếu bạn chưa biết mẹo này."
```

### Caption Template

```
[HOOK — 1 câu scroll-stopper, ≤15 từ]

[BODY — 1-3 key points, mỗi điểm 1 dòng]
✅ Điều 1
✅ Điều 2
✅ Điều 3

[CTA — 1 câu rõ ràng]

[HASHTAGS — 5-10]
```

### Script Template (cho video có voiceover)

```
[0-3s]  HOOK: Câu mở đầu shock/relatable
[3-15s] SETUP: Bối cảnh ngắn gọn (vấn đề gì?)
[15-40s] CONTENT: 2-3 điểm chính, demo/ví dụ
[40-50s] PAYOFF: Kết quả/reveal
[50-60s] CTA: Follow/Save/Comment
```

### CTA Patterns cho TikTok

```
Follow CTA:
- "Follow để xem part 2 🔥"
- "Follow nếu bạn cũng thích mẹo tiết kiệm"

Engagement CTA:
- "Comment '🙋' nếu bạn cũng bị"
- "Bạn team A hay team B?"

Save CTA:
- "Save lại rồi thử ngay nha 📌"
- "Bookmark vì lần tới bạn SẼ cần"

Duet/Stitch CTA:
- "Stitch video này nếu bạn có mẹo hay hơn"
- "Duet và show kết quả của bạn 👇"
```

### Hashtag Strategy VN

```
Tầng 1 (1-2): Trend lớn → #xuhuong #trending #fyp #viral
Tầng 2 (2-3): Niche → #skincarevn #marketingvn #reviewsach
Tầng 3 (2-3): Long-tail → #meotiemkiemtien #hoclamnail #recipevietnam
Tầng 4 (1-2): Branded → #tenbrand #tenchienhdich
```

### Voice & Tone

```
- Viết như ĐANG NÓI — không viết văn
- Slang VN OK: "slay", "real", "vibe", "xỉu", "iu"
- Câu ngắn. Cực ngắn. Như này.
- Humor > Formal
- Self-deprecating OK
- KHÔNG giảng dạy, CHIA SẺ
```

## Anti-AI Rules cho TikTok

```
1. KHÔNG dùng: "Bạn có biết rằng..." / "Một số người cho rằng..."
2. PHẢI có: Ít nhất 1 emoji reaction tự nhiên
3. PHẢI nghe: Như 1 người thật đang nói chuyện qua video
4. Câu ≤10 từ chiếm ≥50% content
5. Tránh: Perfect grammar — imperfection = authentic
```
