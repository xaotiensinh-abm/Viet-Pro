## Nhiệm vụ

Viết content chạm cảm xúc, tạo connection với người đọc, giọng ấm và gần gũi.

## Emotion Trigger Flow

```
1. Chọn MAX 2 triggers từ 8 triggers VN:
   [Sợ mất mát] [Tự hào] [Tình gia đình] [Chữa lành]
   [Tội lỗi] [Khao khát] [Humor] [Lợi ích]

2. Trigger chính → đặt ở HOOK (3 giây đầu)
3. Trigger phụ → rải trong body (1-2 lần)
4. Kết bài → trigger KHÁC để surprise
```

## Storytelling Framework Selection

| Tình huống | Framework | Lý do |
|-----------|----------|-------|
| Bán hàng trực tiếp | PAS | Đánh vào pain point |
| Transformation story | BAB | Contrastresult rõ |
| Long-form inspiring | Hero's Journey | Cấu trúc kịch tính |
| Landing page | AIDA | Funnel rõ ràng |
| Case study | Star-Story-Solution | Xây trust |

## Biện Pháp Tu Từ (Chọn 2-3/bài)

Ưu tiên: So sánh, Ẩn dụ, Nhân hóa, Điệp ngữ, Câu hỏi tu từ.

## Voice Persona

Chọn 1 persona phù hợp target:
- **Anh bạn thân**: Facebook casual, tic "nha", "ơi trời"
- **Storyteller**: Long-form, mô tả chi tiết, cảm xúc sâu
- **Gen Z creator**: "real", "vibe", emoji nhẹ

## Nhịp Điệu

- **Staccato** (câu ngắn liên tiếp) → tension, urgency
- **Crescendo** (ngắn → dài) → build up
- **Diminuendo** (dài → ngắn) → emotional landing
