# Module H7: Platform — Zalo Official Account

> **Load**: Target Zalo OA, Zalo broadcast, Zalo mini-app copy
> **Knowledge**: `knowledge/social-media-vn-2025.md`

---

## Nhiệm vụ

Tạo content tối ưu cho nền tảng Zalo — nền tảng lớn nhất VN (70M+ users).

## Đặc Thù Zalo vs Facebook

```
Zalo ≠ Facebook:
- Zalo = Messaging-first → content ngắn hơn, trực tiếp hơn
- OA = kênh 1:1 với followers → personal, không viral-focused
- Broadcast = SMS marketing nhưng miễn phí → respect inbox
- User VN mọi lứa tuổi (20-55) → tone phổ thông, ít slang
- Ít emoji hơn FB, chuyên nghiệp hơn nhưng vẫn thân thiện
```

## Format Rules

### Zalo OA Article

```
Tiêu đề:  ≤60 ký tự, rõ ràng, KHÔNG clickbait
Mô tả:    ≤120 ký tự, tóm gọn value
Nội dung:  300-800 từ (ngắn hơn blog)
Hình ảnh:  16:9 cover, ≤1MB, text ≤20% diện tích
CTA:       1 CTA duy nhất, rõ ràng
Link:      Ưu tiên internal (Zalo mini-app)
```

### Zalo Broadcast Message

```
Tối đa:    ≤2000 ký tự (nhưng nên ≤500 cho optimal)
Cấu trúc:
  Dòng 1:  Greeting cá nhân "Chào anh/chị [tên],"
  Dòng 2-3: Value proposition ngắn gọn
  Dòng 4:  1 CTA duy nhất (link/button)
  Dòng 5:  Sign-off lịch sự

Tần suất:  ≤2 lần/tuần — SPAM = mất followers
Timing:    9h-11h (sáng) hoặc 14h-16h (chiều)
           TRÁNH: Tối muộn, cuối tuần
```

## Hook Patterns cho Zalo

```
1. PERSONAL: "Anh/chị ơi, [vấn đề] này có quen không?"
2. VALUE:    "[Benefit cụ thể] — chỉ cần 3 phút đọc"
3. URGENCY:  "Ưu đãi đặc biệt cho anh/chị — hết trong [thời gian]"
4. NEWS:     "Tin mới: [sự kiện] —影 hưởng trực tiếp đến..."
5. QUESTION: "Anh/chị đã thử [giải pháp] chưa?"
```

## Voice Persona — Zalo

```
Tone:       Lịch sự + Thân thiện (như nhân viên chăm sóc khách hàng giỏi)
Xưng hô:   "Anh/chị" (mặc định) | "Bạn" (nếu Gen Z/community)
Emoji:      0-2 emoji tối đa, chọn professional (✅ 📌 👉)
Slang:      KHÔNG dùng slang/Gen Z
Language:   Tiếng Việt chuẩn, KHÔNG mix English (trừ brand terms)
```

## Template: Zalo OA Article

```markdown
## [Headline — Max 60 chars]

[Cover image 16:9]

[Opening 1-2 câu — nêu vấn đề/context]

[Body content — 3-5 paragraphs ngắn gọn]
- Bullet points cho key info
- Bold cho từ khóa quan trọng
- Giữ đoạn ≤3 câu

[CTA rõ ràng]
👉 [Link / Button text]

---
[Brand sign-off]
```

## Template: Zalo Broadcast

```
Chào anh/chị [Tên],

[1 câu value proposition — tại sao nên đọc]

[2-3 câu nội dung chính — ngắn gọn, có data nếu có]

👉 [CTA + link]

Cảm ơn anh/chị,
[Brand name]
```

## Anti-AI Rules — Zalo

```
1. KHÔNG academic — Zalo = đời thường, KHÔNG "bối cảnh hiện nay"
2. KHÔNG quá informal — Zalo ≠ Facebook comment, giữ lịch sự
3. CÁ NHÂN HÓA — Luôn có greeting, sign-off, tone 1:1
4. NGẮN GỌN — Zalo users đọc nhanh, KHÔNG viết dài dòng
5. 1 CTA — Quá nhiều CTA = confused = exit
```
