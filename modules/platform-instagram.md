# Module H8: Platform — Instagram

> **Load**: Target Instagram (Feed, Reels, Stories, Carousel)
> **Knowledge**: `knowledge/social-media-vn-2025.md`

---

## Nhiệm vụ

Tạo content tối ưu cho Instagram — nền tảng visual-first, khác biệt rõ với Facebook & TikTok.

## Instagram vs Facebook vs TikTok

```
Instagram ≠ Facebook:
- Visual FIRST → caption phụ trợ cho hình/video
- Hashtag = nền tảng discovery (15-25 optimal)
- Aesthetic consistency = brand identity
- Stories = engagement hub, Feed = portfolio

Instagram ≠ TikTok:
- Curated > Raw (ngược TikTok)
- Longer content lifecycle (Feed posts sống lâu hơn)
- Carousel = engagement king (save + share)
- IG audience VN: 18-35, urban, income trung-cao
```

## Format Rules

### Feed Post Caption

```
Length:    150-300 từ (optimal engagement VN)
Hook:     Dòng đầu = scroll-stopper (hiện trước "Xem thêm")
Body:     Break lines rõ ràng, dùng emoji spacers
CTA:      Cuối caption — comment/save/share
Hashtag:  15-25, mix: 5 trending + 10 niche + 5 branded
Location: LUÔN tag location (boost reach)
```

### Carousel Post

```
Slides:   5-10 (optimal: 7-8)
Slide 1:  HOOK — title slide, bold, gây tò mò
Slide 2-N: Content slides — 1 ý/slide, font lớn
Slide cuối: CTA + brand info + "Save 📌 Share ↗️"

Text/Slide: ≤30 từ, font ≥24pt visual
Format:     1080x1080 hoặc 1080x1350 (portrait = more real estate)
Caption:    Summary + CTA + hashtags
```

### Reels Caption

```
Length:    50-150 từ (ngắn hơn Feed post)
Hook:     1 dòng scroll-stopper
Body:     1-2 dòng context
CTA:      "Follow để xem thêm" / "Save rồi xem lại 📌"
Hashtag:  10-15, niche-focused
Audio:    Gợi ý trending sound nếu relevant
```

### Stories Copy

```
Poll:      2 options, clear và fun
Question:  1 câu hỏi mở (engagement)
Quiz:      3-4 options, 1 correct
Slide:     ≤15 từ text overlay
CTA:       Swipe up / Link sticker (≥10K followers)
Stickers:  Location, mention, hashtag (1-3)
```

## Hook Patterns — Instagram

```
1. CURIOSITY GAP: "Mình đã [kết quả wow] chỉ bằng 1 thay đổi nhỏ..."
2. BOLD CLAIM:    "Tin mình đi: [controversial take]"
3. LISTICLE:      "5 điều mình ước biết sớm hơn về [topic]"
4. BEFORE/AFTER:  "Slide 1 = before → Slide cuối = 🤯"
5. QUESTION:      "Bạn team A hay team B? 👇"
6. STORY OPEN:    "Hôm qua mình gặp chuyện này..."
```

## Hashtag Strategy VN — Instagram

```
Tầng 1 — BIG (5):       #trending #viral #vietnam #instagood #photooftheday
Tầng 2 — INDUSTRY (5-7): #marketingvn #startupvietnam #designvn #lifestylevn
Tầng 3 — NICHE (5-8):    #contentcreatovn #saigonfood #reviewhanoi
Tầng 4 — BRANDED (2-3):  #tenbrand #tenchienhdich

Placement: Cuối caption HOẶC first comment
Rotate: Thay đổi 30-50% hashtags mỗi post để tránh shadowban
```

## Voice Persona — Instagram VN

```
Tone:       Personal, curated, aspirational nhưng relatable
Xưng hô:   "Mình" / "Bạn" (casual, friendly)
Emoji:      3-5 per caption, decorative + functional
Language:   Mix VN + EN trendy OK ("vibe", "mood", "aesthetic")
Aesthetic:  Nhất quán visual theme — warm tones, clean layout
```

```
📌 Save lại để xem khi cần nha~

```

## Anti-AI Rules — Instagram

```
1. VISUAL-FIRST: Caption PHỤ TRỢ cho visual, KHÔNG thay thế
2. CURATED CASUAL: Polished nhưng không academic
3. EMOJI NATURAL: Dùng để break text + emotion, KHÔNG spam
4. HASHTAG RELEVANT: Chỉ dùng hashtag liên quan, KHÔNG fill random
5. PERSONAL VOICE: "Mình" perspective, KHÔNG "chúng ta" generic
6. NO ESSAY: Instagram ≠ blog, giữ caption ≤300 từ
```
