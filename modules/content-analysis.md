# Module N2: Content — Analysis & Structure

> **Load**: Long-form, phân tích chuyên sâu, series content

---

## Nhiệm vụ

Phân tích thông tin từ N1, lựa chọn framework, tạo outline cấu trúc.

## Analysis Process

```
1. CLUSTER → Nhóm insights theo theme
2. PRIORITIZE → Chọn 3-5 ý mạnh nhất (bỏ phần yếu)
3. SEQUENCE → Sắp xếp logic: hook → build → climax → landing
4. FRAMEWORK → Chọn framework phù hợp
5. OUTLINE → Tạo skeleton cho bài viết
```

## Framework Selection Matrix

| Loại bài | Framework chính | Framework phụ |
|---------|----------------|-------------|
| Persuasion/Sales | PAS | AIDA |
| Transformation | BAB | Hero's Journey |
| Analysis/Opinion | Commentary Formula | Reframing |
| Howto/Tutorial | Step-by-step | ELI15 |
| Listicle | Parallel structure | — |
| Case study | Star-Story-Solution | BAB |
| Newsletter | Compounding Ideas | Commentary |

## Long-form Pacing

| Phần | % bài | Tốc độ | Kỹ thuật |
|------|------|-------|---------|
| Hook | 5% | Fast/shock | Câu ngắn, stat, question |
| Context | 15% | Medium | Bối cảnh, bridge |
| Body 1 | 25% | Deep | Analysis, evidence |
| Body 2 | 25% | Deep + emotion | Story, metaphor |
| Body 3 | 15% | Medium | Synthesis |
| Closing | 15% | Slow → punch | Emotional landing |
