## Bloom's Taxonomy — 6 Cấp Tư Duy

### Pyramid (Thấp → Cao)

```
┌────────────────────────┐
│  6. SÁNG TẠO (Create)  │  Thiết kế, xây dựng, đề xuất giải pháp mới
├────────────────────────┤
│  5. ĐÁNH GIÁ (Evaluate)│  Phê bình, so sánh, biện luận, quyết định
├────────────────────────┤
│  4. PHÂN TÍCH (Analyze) │  Phân biệt, tổ chức, liên kết, tìm pattern
├────────────────────────┤
│  3. VẬN DỤNG (Apply)   │  Thực hiện, giải quyết, sử dụng trong tình huống mới
├────────────────────────┤
│  2. HIỂU (Understand)  │  Giải thích, mô tả, tóm tắt, cho ví dụ
├────────────────────────┤
│  1. NHỚ (Remember)     │  Nhận diện, liệt kê, định nghĩa, nhắc lại
└────────────────────────┘
```

### Hành Động Từ (Action Verbs) theo Cấp — Tiếng Việt

| Cấp | Hành động từ | Hoạt động phù hợp |
|-----|-------------|-------------------|
| **1. Nhớ** | Liệt kê, xác định, ghi nhớ, nhận diện, nhắc lại, kể tên | Flashcard, danh sách, quiz recall, matching |
| **2. Hiểu** | Giải thích, mô tả, tóm tắt, phân loại, diễn giải, cho ví dụ | Paraphrase, summary, concept map, Q&A |
| **3. Vận dụng** | Áp dụng, thực hiện, tính toán, giải quyết, chứng minh, sử dụng | Bài tập, tình huống, role-play, simulation |
| **4. Phân tích** | So sánh, phân biệt, tổ chức, liên kết, tìm nguyên nhân | SWOT, case analysis, compare-contrast, debate |
| **5. Đánh giá** | Phê bình, biện luận, đánh giá, chọn lọc, phản biện, bảo vệ | Peer review, rubric assessment, defend position |
| **6. Sáng tạo** | Thiết kế, xây dựng, đề xuất, phát triển, sáng tác, lập kế hoạch | Project, design thinking, proposal, prototype |

### Ví Dụ Mục Tiêu Theo Bloom — Chủ Đề "Marketing Digital"

```
L1 Nhớ:     Liệt kê được 5 kênh marketing digital phổ biến tại VN
L2 Hiểu:    Giải thích sự khác biệt giữa SEO và SEM
L3 Vận dụng: Lập kế hoạch content calendar 1 tuần cho fanpage
L4 Phân tích: So sánh hiệu quả Facebook Ads vs Google Ads cho ngành F&B
L5 Đánh giá: Đánh giá chiến lược marketing của Shopee VN Q4/2025
L6 Sáng tạo: Thiết kế chiến dịch marketing tích hợp cho startup EdTech
```

## Viết Mục Tiêu Học Tập (Learning Objectives)

### Công Thức ABCD

```
A = Audience:   Ai học? (Sinh viên năm 3 ngành Marketing)
B = Behavior:   Làm được gì? (Hành động từ Bloom)
C = Condition:  Trong điều kiện nào? (Dùng Excel, trong 30 phút)
D = Degree:     Đạt mức nào? (Chính xác 80%, ít nhất 3/5)
```

### Ví Dụ ABCD

```
❌ Yếu: "Hiểu về kế toán"
✅ Tốt: "Sau bài học, sinh viên (A) có khả năng hạch toán (B) 
         5 nghiệp vụ kế toán cơ bản trên Excel (C) 
         với độ chính xác ≥90% (D)."
```

### Checklist Mục Tiêu Tốt

```
☐ Bắt đầu bằng hành động từ (KHÔNG dùng: "biết", "nắm được")
☐ Observable & Measurable (đo được)
☐ Align với Bloom level phù hợp
☐ Specific — không mơ hồ
☐ Achievable trong thời lượng cho phép
```

## Instructional Design Patterns

### Pattern 1: BOPPPS (Phổ biến cho bài giảng)

```
B = Bridge-in:    Hook, kết nối với kiến thức đã có
O = Objectives:   Nêu rõ mục tiêu, "Cuối buổi bạn sẽ..."
P = Pre-test:     Kiểm tra kiến thức nền (quick quiz, show of hands)
P = Participatory: Hoạt động học tập chính (không chỉ nghe)
P = Post-test:    Đánh giá đạt mục tiêu chưa (quiz, exit ticket)
S = Summary:      Tóm tắt + kết nối bài sau
```

### Pattern 2: 5E (Inquiry-based, STEM)

```
Engage:    Kích thích tò mò (câu hỏi, video, demo)
Explore:   Khám phá tự do (thí nghiệm, nhóm, tìm hiểu)
Explain:   Giải thích lý thuyết (sau khi đã explore)
Elaborate: Mở rộng, ứng dụng (bài tập nâng cao, dự án)
Evaluate:  Đánh giá (quiz, reflection, peer review)
```

### Pattern 3: Flipped Classroom

```
TRƯỚC BUỔI HỌC:
- Video bài giảng (5-15 phút)
- Reading assignment
- Pre-quiz (kiểm tra đã xem/đọc chưa)

TRONG BUỔI HỌC:
- KHÔNG giảng lại lý thuyết
- Thảo luận, Q&A, làm bài tập
- Group project, case study
- Giảng viên = facilitator

SAU BUỔI HỌC:
- Reflection journal
- Extension activities
```

### Pattern 4: Problem-Based Learning (PBL)

```
1. Đưa VẤN ĐỀ thực tế trước (chưa có kiến thức mới)
2. Học sinh TỰ xác định cần biết gì để giải quyết
3. Tự nghiên cứu + tham khảo tài liệu
4. Đề xuất giải pháp
5. Phản biện + đánh giá lẫn nhau
6. Giảng viên tổng hợp, bổ sung kiến thức
```

## Active Learning Techniques

### Engagement Toolkit (Chèn mỗi 500 từ)

| Kỹ thuật | Thời gian | Bloom Level | Mô tả |
|----------|-----------|------------|-------|
| Think-Pair-Share | 3-5' | L2-4 | Nghĩ 1 phút → Trao đổi cặp → Chia sẻ lớp |
| Exit Ticket | 2-3' | L1-3 | Viết 1 điều học được + 1 câu hỏi còn thắc mắc |
| Muddiest Point | 1-2' | L2 | "Điều gì bạn thấy khó hiểu nhất?" |
| Quick Write | 3-5' | L3-5 | Viết tự do 3 phút về chủ đề vừa học |
| Gallery Walk | 10-15' | L4-6 | Nhóm trình bày poster → nhóm khác comment |
| Jigsaw | 15-20' | L3-5 | Mỗi nhóm chuyên gia 1 phần → dạy lại nhóm mới |
| Case Study Mini | 5-10' | L4-5 | Tình huống ngắn → "Bạn sẽ làm gì?" |
| Debate | 10-15' | L5-6 | 2 phe bảo vệ quan điểm, có rubric |

### Assessment Types

```
Formative (Trong quá trình):
- Quiz ngắn (5 câu, 3 phút)
- Bài tập nhóm mini
- Phản hồi miệng, show of hands
- Concept map vẽ nhanh

Summative (Cuối unit/khóa):
- Bài kiểm tra (trắc nghiệm + tự luận)
- Dự án cuối kỳ
- Portfolio
- Thuyết trình
```

## Vietnamese Education Context

### Bối Cảnh Giáo Dục VN

```
1. Hệ thống: 12 năm phổ thông + 4 năm đại học (15 tín chỉ/kỳ trung bình)
2. Xu hướng: Chuyển đổi từ truyền thụ → năng lực (Chương trình 2018)
3. Tech: Zoom/Teams phổ biến, LMS (Moodle, Google Classroom)
4. Văn hóa: Tôn trọng thầy cô, học nhóm phổ biến, thi cử áp lực
5. Assessment: Thang điểm 10 (phổ thông) hoặc 4.0 GPA (đại học)
```

### Localization Rules

```
1. VÍ DỤ: Dùng tên doanh nghiệp VN, số liệu thống kê VN (GSO)
2. PHÁP LUẬT: Tham chiếu luật VN khi relevant (Luật Doanh nghiệp, Luật Lao động)
3. VĂN HÓA: Tình huống phù hợp văn hóa VN (gia đình, cộng đồng, quan hệ)
4. NGÔN NGỮ: Thuật ngữ chuyên ngành giữ tiếng Anh + giải thích tiếng Việt
5. THỰC TẾ: Case study từ FPT, Vingroup, VinaSoy, Thế Giới Di Động...
```

## Mnemonic Devices — Kỹ Thuật Ghi Nhớ

```
1. Viết tắt: SMART (Specific, Measurable, Achievable, Relevant, Time-bound)
2. Câu vần: "Có công mài sắt, có ngày nên kim" → tie vào concept
3. Story method: Gắn kiến thức vào câu chuyện
4. Visual: Sơ đồ, mindmap, infographic
5. Chunking: Nhóm thông tin thành clusters 3-5 items
6. Elaboration: Liên hệ kiến thức mới với kiến thức đã có
```
