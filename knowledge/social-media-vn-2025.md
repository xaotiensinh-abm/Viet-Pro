## Platform-Specific Data

### Facebook VN

```
Algorithm Priority 2025:
1. Original content > Shared content
2. Video/Reels > Image > Text-only
3. Meaningful interactions (comments > reactions > shares)
4. Group engagement tăng trọng số
5. Creator content được boost (Creator Mode)

Peak Posting Times VN:
- B2C: 11h30-13h30 (lunch) | 20h-22h (tối)
- B2B: 8h-9h (sáng) | 14h-15h (chiều)
- Weekend: 9h-11h (sáng chủ nhật)

Engagement Benchmarks VN:
- Engagement rate trung bình: 0.5-1.5%
- Video view rate: 3-8% followers
- Reach organic: 3-8% followers (giảm mạnh)
- Share = metric có giá trị nhất (boost reach 5-10x)
```

### TikTok VN

```
Algorithm Priority 2025:
1. Watch time / Completion rate (XEM XONG = KING)
2. Shares > Saves > Comments > Likes
3. First 3 seconds quyết định everything
4. Trending sounds boost discovery
5. Niche hashtag > Generic hashtag

Peak Posting Times VN:
- 7h-9h (commute) | 12h-13h (lunch) | 19h-22h (tối)
- Thứ 7: 10h-12h, 19h-22h (peak weekend)

Content Trends VN 2025:
- "Day in my life" / GRWM (Get Ready With Me)
- Review sản phẩm thật, không filter
- "Mẹo tiết kiệm" / "Life hack VN"
- Storytelling format (chia part 1, 2, 3)
- Educational content ngắn gọn (Edutok)
- "Trước vs Sau" transformation
```

### LinkedIn VN

```
Algorithm Priority 2025:
1. Dwell time (thời gian đọc) > Click
2. Comments > Reactions
3. Native content > External links
4. Document/Carousel post cao engagement nhất
5. Personal profile > Company page reach

Peak Posting Times VN:
- T2-T6: 7h30-8h30 (before work) | 12h-13h (lunch)
- T3, T4 có engagement cao nhất
- Tránh: T7, CN (engagement thấp)

Content Trends VN:
- Founder/CEO storytelling
- Career advice / Lessons learned
- Industry insights + data VN
- "Tôi fail/sai ở đây" (vulnerability = gold)
- Document/Carousel how-to guides
```

### YouTube VN

```
Algorithm Priority 2025:
1. Click-through rate (CTR) thumbnail + title
2. Watch time (tổng & %) — retention curve
3. Session time (kéo user ở lại YouTube)
4. Engagement: likes, comments, shares
5. Shorts: completion rate + replay

Content Trends VN:
- YouTube Shorts tăng mạnh (cạnh tranh TikTok)
- Podcast video format phổ biến
- "Giải thích" / "Sự thật về..." (educational)
- Reaction / Commentary Vietnamese style
```

## Content Trends Chung VN 2025-2026

### Macro Trends

```
1. SHORT-FORM FIRST: Video ≤60s dominate discovery
2. AUTHENTICITY: "Real > Perfect" — unfiltered content thắng
3. AI TOOLS: Người dùng VN tăng dùng AI tạo content
4. SOCIAL COMMERCE: Mua hàng trực tiếp trên social (TikTok Shop, FB Marketplace)
5. COMMUNITY: Group/Community engagement > Broadcast content
6. CREATOR ECONOMY: KOLs nhỏ (1k-50k followers) hiệu quả hơn KOLs lớn
7. LOCALIZATION: Content tiếng Việt, văn hóa VN, humor VN
8. UGC: User-generated content = social proof mạnh nhất
```

### Content Format Preferences (VN Audience)

```
Ưu tiên cao:
✅ Video ngắn (TikTok, Reels, Shorts)
✅ Carousel/Image series (Instagram, LinkedIn)
✅ Storytelling posts (Facebook, LinkedIn)
✅ Interactive (Poll, Quiz, Q&A)

Ưu tiên trung bình:
⚠️ Long-form article (Blog, LinkedIn)
⚠️ Podcast (YouTube, Spotify)
⚠️ Email newsletter

Ưu tiên thấp:
⬇️ Text-only posts (trừ Twitter)
⬇️ Infographic tĩnh (quá cũ)
⬇️ Webinar dài (>60 phút)
```

## Vietnamese Social Media Language

### Slang & Trends 2025

```
Gen Z VN:
- "slay" = làm tốt, ấn tượng
- "real" = thật, đáng tin
- "vibe" = cảm giác, không khí
- "xỉu" = quá yêu thích
- "iu" = yêu (viết tắt)
- "ngl" = not gonna lie = thật lòng mà nói
- "no cap" = không nói đùa
- "delulu" = delusion = ảo tưởng (hài)
- "toxic" = độc hại
- "chill" = thư giãn, nhẹ nhàng

All ages:
- "Ơ xời" = bất ngờ nhẹ
- "Trời ơi" = surprise/disbelief
- "Quá xịn" = rất tốt
- "Khó đỡ" = dở hơi / buồn cười
- "Cực kỳ" = very (nhấn mạnh)
```

### Platform Language Map

```
Facebook:   Mix formal + casual, emoji vừa phải, tiếng Việt chuẩn
LinkedIn:   Professional, ít emoji, data-driven, tiếng Việt formal
TikTok:     Gen Z slang OK, viết như nói, punchy, 100% casual
Twitter:    Concise, witty, opinion mạnh, tiếng Việt tự nhiên
Instagram:  Visual-first, caption nhẹ, hashtag-heavy
Email:      Personal, warm, professional nhưng không stiff
Blog:       Semi-formal, authoritative, SEO-optimized
```
