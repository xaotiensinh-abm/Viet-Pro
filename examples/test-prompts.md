# 🧪 Test Prompts — Kiểm Tra Mọi Pipeline

> Dùng file này để verify skill hoạt động đúng cho từng pipeline.

---

## Pipeline A: Novel Writer (Fiction)

```
/novel init
→ Thể loại: Đô Thị Tu Tiên
→ Outline arc 1 (5 chapters)
→ Viết chapter 1: MC thức tỉnh hệ thống trong phòng trọ Sài Gòn

Expect: 3000-5000 từ, QA score ≥85/130, burstiness, VN soul
```

---

## Pipeline B: Article/Blog (Viet-Pro Swarm)

```
Viết bài Facebook 500 từ về "3 sai lầm phổ biến khi dùng AI trong kinh doanh"
→ Audience: Chủ doanh nghiệp SME
→ Tone: Thân thiện + chuyên gia
→ CTA: Comment chia sẻ trải nghiệm

Expect: PAS/BAB framework, anti-AI ≥7/10, thành ngữ VN, humor
```

## Pipeline C: EduWriter

```
Soạn lesson plan cho bài "Content Marketing cơ bản"
→ Đối tượng: Sinh viên Marketing năm 3
→ Thời lượng: 90 phút
→ Bloom level: Understand + Apply
→ Format: BOPPPS model

Expect: Mục tiêu với Bloom verbs, 5E/BOPPPS structure, active learning activities
```

## Pipeline D: SlideWriter

## Pipeline E: ResearchWriter

```
Viết literature review 2000 từ về "Chuyển đổi số doanh nghiệp nhỏ và vừa Việt Nam"
→ Format: IMRaD
→ Citations: APA 7th
→ Sources: ≥10 (Tier 1-2)

Expect: Academic tone, hedging language VN, proper citation format, CRAAP-verified
```

## Pipeline F: SocialWriter

